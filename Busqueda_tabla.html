Palabra Definición
HTML Es el lenguaje de marcado utilizado para crear páginas web.
CSS Es un lenguaje utilizado para dar estilo a las páginas web.
JavaScript Es un lenguaje de programación utilizado para crear contenido interactivo en la web.
Python Es un lenguaje de programación popular para desarrollo web, ciencia de datos, inteligencia artificial, entre otros.
Java Es un lenguaje de programación de propósito general, usado principalmente en aplicaciones empresariales.
Dom Es una representación estructurada de un documento HTML o XML que permite interactuar con sus elementos a través de JavaScript para modificar su contenido, estructura o estilo.
